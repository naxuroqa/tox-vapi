audio_set_bitrate

Description:

[ Version ( since = "0.2.0" ) ]
public bool audio_set_bitrate (uint32 friend_number, uint32 bit_rate, ref ErrBitRateSet error)

Set the bit rate to be used in subsequent video frames.

Parameters:

friend_number

The friend number of the friend for which to set the bit rate.

bit_rate

The new audio bit rate in Kb/sec. Set to 0 to disable.

Returns:

true on success.

0.2.0
