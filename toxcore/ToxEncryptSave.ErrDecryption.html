ErrDecryption

Description:

public enum ErrDecryption

Namespace: ToxEncryptSave
Package: toxcore

Content:

Enum values:

  • OK - The function returned successfully.
  • NULL - One of the arguments to the function was NULL when it was not expected.
  • INVALID_LENGTH - The input data was shorter than pass_encryption_extra_length bytes
  • BAD_FORMAT - The input data is missing the magic number (i.e. wasn't created by this module, or is corrupted).
  • KEY_DERIVATION_FAILED - The crypto lib was unable to derive a key from the given passphrase, which is usually a lack of memory issue. The functions accepting keys do not produce this error.
  • FAILED - The encrypted byte array could not be decrypted. Either the data was corrupted or the password/key was incorrect.
