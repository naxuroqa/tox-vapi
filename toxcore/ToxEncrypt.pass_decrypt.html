pass_decrypt

Description:

public uint8[]? pass_decrypt (uint8[] ciphertext, uint8[]? passphrase, ref ErrDecryption error)

Decrypts the given data with the given passphrase.

The output array must be at least `ciphertext_len - TOX_PASS_ENCRYPTION_EXTRA_LENGTH` bytes long. This delegates to tox_pass_key_decrypt.

Parameters:

ciphertext

A byte array of length `ciphertext_len`.

passphrase

The user-provided password. Can be empty.

Returns:

plaintext on success.
