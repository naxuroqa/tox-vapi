ErrAnswer

Description:

public enum ErrAnswer

Namespace: ToxAV
Package: toxcore

Content:

Enum values:

  • OK - The function returned successfully.
  • SYNC - Synchronization error occurred.
  • CODEC_INITIALIZATION - Failed to initialize codecs for call session. Note that codec initiation will fail if there is no receive callback registered for either audio or video.
  • FRIEND_NOT_FOUND - The friend number did not designate a valid friend.
  • FRIEND_NOT_CALLING - The friend was valid, but they are not currently trying to initiate a call. This is also returned if this client is already in a call with the friend.
  • INVALID_BIT_RATE - Audio or video bit rate is invalid.
