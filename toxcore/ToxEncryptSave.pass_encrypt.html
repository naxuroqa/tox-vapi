pass_encrypt

Description:

public uint8[]? pass_encrypt (uint8[] plaintext, uint8[]? passphrase, ref ErrEncryption error)

Encrypts the given data with the given passphrase.

The output array must be at least `plaintext_len + TOX_PASS_ENCRYPTION_EXTRA_LENGTH` bytes long. This delegates to tox_pass_key_derive and tox_pass_key_encrypt.

Parameters:

plaintext

A byte array of length `plaintext_len`.

passphrase

The user-provided password. Can be empty.

Returns:

ciphertext on success.
