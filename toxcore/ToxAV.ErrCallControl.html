Enum values:

  • FRIEND_NOT_FOUND - The friend_number passed did not designate a valid friend.
  • FRIEND_NOT_IN_CALL - This client is currently not in a call with the friend. Before the call is answered, only CANCEL is a valid control.
  • INVALID_TRANSITION - Happens if user tried to pause an already paused call or if trying to resume a call that is not paused.
  • OK - The function returned successfully.
  • SYNC - Synchronization error occurred.
