CallControl

Description:

public enum CallControl

Namespace: ToxAV
Package: toxcore

Content:

Enum values:

  • RESUME - Resume a previously paused call. Only valid if the pause was caused by this client, if not, this control is ignored. Not valid before the call is accepted.
  • PAUSE - Put a call on hold. Not valid before the call is accepted.
  • CANCEL - Reject a call if it was not answered, yet. Cancel a call after it was answered.
  • MUTE_AUDIO - Request that the friend stops sending audio. Regardless of the friend's compliance, this will cause the audio_receive_frame event to stop being triggered on receiving an audio frame from the friend.
  • UNMUTE_AUDIO - Calling this control will notify client to start sending audio again.
  • HIDE_VIDEO - Request that the friend stops sending video. Regardless of the friend's compliance, this will cause the video_receive_frame event to stop being triggered on receiving a video frame from the friend.
  • SHOW_VIDEO - Calling this control will notify client to start sending video again.
