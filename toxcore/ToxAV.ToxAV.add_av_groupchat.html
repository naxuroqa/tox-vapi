add_av_groupchat

Description:

public static int add_av_groupchat (Tox tox, GroupchatAudioReceiveFrameCallback callback)

Create a new toxav group.

Audio data callback format: audio_callback(Tox *tox, uint32_t groupnumber, uint32_t peernumber, const int16_t *pcm, unsigned int samples, uint8_t channels, uint32_t sample_rate, void *userdata)

Note that total size of pcm in bytes is equal to (samples * channels * sizeof(int16_t)).

Returns:

group number on success.

-1 on failure.
